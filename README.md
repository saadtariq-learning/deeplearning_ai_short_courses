## 📘 DeepLearning.AI – Short Courses Collection

A curated collection of DeepLearning.AI short courses, organized into individual folders for easy navigation, practice, and revision.
This repository is ideal for learners looking to master modern AI, ML, and Generative AI concepts through hands-on notebooks and structured lessons.

## 📂 Repository Structure

Each folder represents one DeepLearning.AI short course and typically contains:

- 📓 Jupyter notebooks
- 🧠 Concept notes
- 🧪 Exercises & experiments
- 📄 Supporting resources (if any)
